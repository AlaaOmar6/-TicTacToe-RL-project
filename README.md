# -TicTacToe-RL-project
Tic-Tac-Toe Reinforcement Learning - Project 2. Comparing Minimax, Dynamic Programming, Monte Carlo, and Temporal Difference (Q-Learning &amp; SARSA) on a 3x3 Tic-Tac-Toe game. Implements state space verification (2423 non-terminal X-player states), DP-Minimax consistency checks, and epsilon-decay exploration
